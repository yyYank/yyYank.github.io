# KotlinをNetBeansで使う

* インストール
NetBeans 8.1でベータversionのKotlinプラグインが入手出来ます。

## Installation process:

* NetBeansを起動します
* Tools>Pluginsをmenuから選択
* Downloadedタブを選択
* Add Plugins…ボタンをクリック
* エクスプローラでダウンロードされたファイルを選択
* Pluginsダイアログでインストールボタンを押す
* Nextを押しライセンスに同意する
