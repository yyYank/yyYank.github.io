## インストール中の一覧見る

```bash
brew list
```

## cleanup

```bash
brew cleanup
```

## upgrade

```bash
brew upgrade hogehoge
```

## uninstall

```bash
brew uninstall hogehoge
```

## install

```bash
brew install hogehoge
```

## 依存関係見る

```bash
brew deps --tree hogehoge
```

## brewおかしいなって時

```bash
brew doctor
```
